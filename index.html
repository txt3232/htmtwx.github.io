Как сделать Termux с XFCE

⚠️ ПРЕДУПРЕЖДЕНИЯ ПЕРЕД НАЧАЛОМ

1. Нужен Android 8+ (лучше Android 10+)

2. Не менее 3 ГБ свободного места

3. Стабильная интернет соединение (WI-FI лучше)

4. Зарядка 50% + (процесс энергоемкий)

5. Терпение

Установка Termux

Termux либо скачиваем с F-Droid либо с GitHub

Termux

Дайте termux все разрешения и отключите оптимизацию батареи

Скачиваем apk файл Termux и заходим

Первым делом вводим pkg install unzip git Это скачает программу для распоковки zip архивов и скачает программу для GitHub

Дальше нужен файл install

install

Скачиваем и открываем в Termux

Вводим chmod +x install Это даст разрешение для файла

И запускаем ./install Это запускает программу

Termux:X11

Еще надо скачать Termux:X11

Скачать Termux:X11 нужно с GitHub

Скачиваем apk файл Termux:X11 и перезаходим в Termux

Запуск

Вводим xstart Запускает XFCE

И заходим в Termux:X11

Готово! Теперь у вас есть Termux с XFCE
